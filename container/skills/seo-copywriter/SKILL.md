# SEO Copywriting Guidelines

Follow these rules strictly when generating articles, blog posts, or marketing copy. Do not write generic AI text; write like a human expert.

## Core Principles

1. **Hook the Reader Immediately**
   - The first sentence must be punchy.
   - Address the reader's pain point within the first paragraph.
   - Use the PAS formula (Problem, Agitate, Solution).

2. **SEO Structure**
   - **H1**: Must contain the primary keyword. Keep it under 60 characters.
   - **H2s**: Use logical subheadings. Include secondary keywords naturally.
   - **Paragraphs**: Keep them short (max 3-4 sentences). People skim on mobile.
   - **Bullet Points**: Use lists wherever possible to break up text walls.

3. **Tone and Voice**
   - **Avoid AI Clichés**: NEVER use phrases like "In today's fast-paced digital world", "Unlock the power of", "Dive into", "A tapestry of".
   - **Active Voice**: Use active verbs. (e.g., "The software saves you time" instead of "Time is saved by the software").
   - **Conversational**: Write like you are explaining it to a smart friend.

4. **Formatting**
   - Use **bolding** for key concepts.
   - Include a clear "Call to Action" (CTA) at the very end.

## Example Output Structure

```markdown
# [Catchy Title with Primary Keyword]

[Engaging opening hook. 2-3 sentences max.]

## [H2: The Core Problem or Question]
[Explanation using short paragraphs and active voice.]

## [H2: Key Benefits or Steps]
* **Benefit 1**: [Brief explanation]
* **Benefit 2**: [Brief explanation]
* **Benefit 3**: [Brief explanation]

## Conclusion
[Brief summary]

**[Call to Action]**: [What should the reader do next?]
```
